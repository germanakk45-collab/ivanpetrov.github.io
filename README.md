# AI Hub — Каталог AI-инструментов и статьи про искусственный интеллект

Современный сайт-каталог AI-инструментов с длинными статьями, гайдами и подборками для СНГ-аудитории.

## Структура файлов

```
ai-tools-catalog/
├── index.html              # Главная страница
├── tools.html              # Каталог инструментов
├── tool-detail.html        # Страница отдельного инструмента
├── articles.html           # Раздел статей
├── article-detail.html     # Страница отдельной статьи
├── about.html              # О проекте
├── contact.html            # Telegram и контакты
├── css/
│   └── style.css           # Основные стили
├── js/
│   └── main.js             # Логика: поиск, фильтры, рендеринг
├── data/
│   ├── tools.js            # Данные об AI-инструментах
│   └── articles.js         # Данные о статьях
└── images/                 # Папка для изображений (пока пустая)
```

## Как открыть сайт

### Способ 1: Просто открыть в браузере
1. Распакуйте архив в любую папку
2. Дважды кликните на `index.html`
3. Сайт откроется в браузере

### Способ 2: Через локальный сервер (рекомендуется)
Если у вас установлен Python:
```bash
cd ai-tools-catalog
python -m http.server 8000
```
Затем откройте `http://localhost:8000` в браузере.

### Способ 3: VS Code + Live Server
1. Откройте папку в VS Code
2. Установите расширение "Live Server"
3. Правый клик на `index.html` → "Open with Live Server"

## Как добавлять новые инструменты

Откройте файл `data/tools.js` и добавьте новый объект в массив `toolsData`:

```javascript
{
    id: "уникальный-id",           // Латиницей, без пробелов (например: "my-new-tool")
    name: "Название инструмента",   // Отображаемое название
    shortDesc: "Краткое описание...", // 1-2 предложения
    category: "Текст",               // Одна из категорий (см. ниже)
    price: "Freemium",               // Бесплатно / Freemium / Платно
    priceLabel: "Бесплатно / Pro $10/мес", // Подробнее о цене
    audience: "Для кого подходит...", // Целевая аудитория
    website: "https://example.com",  // Ссылка на сайт
    features: [                      // Основные функции
        "Функция 1",
        "Функция 2",
        "Функция 3"
    ],
    pros: [                          // Плюсы
        "Плюс 1",
        "Плюс 2"
    ],
    cons: [                          // Минусы
        "Минус 1",
        "Минус 2"
    ],
    alternatives: "Альтернатива 1, Альтернатива 2",
    rating: 4.5,                     // Оценка от 1 до 5
    ratingText: "Краткая оценка",    // Текстовая оценка
    popular: true                    // Показывать на главной?
}
```

### Доступные категории:
- Текст, Изображения, Видео, Презентации, Маркетинг, Бизнес, Автоматизация, Дизайн, Код, Обучение, Аудио

## Как добавлять новые статьи

Откройте файл `data/articles.js` и добавьте новый объект в массив `articlesData`:

```javascript
{
    id: "уникальный-id",            // Латиницей (например: "my-new-article")
    title: "Заголовок статьи",
    subtitle: "Подзаголовок / описание",
    category: "Гайды",              // Обзоры / Гайды / Подборки / Сравнения / Личный опыт
    date: "9 мая 2026",
    readTime: "10 минут",
    tags: ["тег1", "тег2", "тег3"],
    coverEmoji: "🎯",               // Эмодзи для обложки
    coverColor: "#4F46E5",          // Цвет фона обложки (hex)
    tldr: "Краткое содержание...",   // Блок "Коротко"
    toc: [                           // Содержание (заголовки)
        "Первый раздел",
        "Второй раздел",
        "Третий раздел"
    ],
    content: `                       // HTML-содержимое статьи
        <h2 id="toc-1">Первый раздел</h2>
        <p>Текст параграфа...</p>
        <ul>
            <li>Пункт 1</li>
            <li>Пункт 2</li>
        </ul>

        <div class="article-tip">
            <strong>Совет:</strong> Полезный совет...
        </div>

        <h2 id="toc-2">Второй раздел</h2>
        <p>Ещё текст...</p>

        <table class="article-table">
            <tr><th>Колонка 1</th><th>Колонка 2</th></tr>
            <tr><td>Данные</td><td>Данные</td></tr>
        </table>
    `,
    conclusion: "Итоговый вывод...",
    relatedTools: ["chatgpt", "perplexity"], // ID связанных инструментов
    similarArticles: ["other-article-id"],    // ID похожих статей
    featured: true                           // Показывать на главной?
}
```

## Что можно менять

### 1. Тексты и ссылки
- **Название сайта**: в `index.html` найдите все вхождения "AI Hub" и замените
- **Telegram-ссылка**: замените `https://t.me/your_channel` во всех HTML-файлах
- **Email**: замените `hello@aihub.ru` во всех HTML-файлах
- **Описания**: меняйте тексты прямо в HTML-файлах

### 2. Цвета
Откройте `css/style.css` и найдите `:root` (начало файла). Там определены все цвета:

```css
:root {
    --color-primary: #4F46E5;        /* Основной синий */
    --color-accent-purple: #7C3AED;  /* Фиолетовый акцент */
    --color-accent-cyan: #0891B2;    /* Бирюзовый акцент */
    --color-telegram: #229ED9;        /* Цвет Telegram */
    --color-bg: #FAFBFC;              /* Фон */
    --color-text: #1A1A2E;            /* Текст */
    /* ... и другие */
}
```

### 3. Карточки инструментов
Добавляйте/удаляйте в `data/tools.js`

### 4. Статьи
Добавляйте/удаляйте в `data/articles.js`

### 5. Подборки на главной
В `index.html` найдите секцию `.collections-grid` и меняйте карточки.

## Технические детали

- **Фреймворки**: Нет. Чистый HTML, CSS, JavaScript
- **Зависимости**: Google Fonts (Inter)
- **Backend**: Нет
- **База данных**: Нет
- **Адаптивность**: Да, работает на всех устройствах
- **Поиск**: Работает на JavaScript (фильтрация по массиву данных)
- **Фильтры**: Работают на JavaScript

## SEO и мета-теги

Каждая страница имеет:
- Уникальный `<title>`
- `<meta name="description">`
- Semantic HTML (article, section, nav, header, footer)
- Чистые URL с параметрами (`?id=chatgpt`)
